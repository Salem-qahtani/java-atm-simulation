# Java ATM Simulation

A console-based ATM simulation written in Java using object-oriented programming principles.

## Features
- Login using account number
- Deposit and withdraw money
- Transfer money between accounts
- Transaction fees
- Account statistics
- Admin panel for system-wide statistics

## Technologies
- Java
- OOP (Encapsulation, Abstraction, Static members)

## How to Run
1. Compile the project
2. Run `ATM_Simulation`
3. Follow the console instructions
